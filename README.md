# mir-blas
ndslice wrapper for BLAS

## Required libraries

See [wiki: Link with CBLAS & LAPACK](https://github.com/libmir/mir-lapack/wiki/Link-with-CBLAS-&-LAPACK).

## Wrapped API

### Level 1
 - dot
 - nrm2
 - asum
 - copy
 - axpy
 - scal
 - swap

### Level 2
 - gemv
 - ger (includes geru)
 - gerc (includes ger)

### Level 3
 - gemm
 - syrk
 - trmm
 - trsm
